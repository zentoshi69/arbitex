# Styrene B Font Files

Place the following font files in this directory for the ArbitEx redesign:

- `StyreneB-Light.woff2` or `StyreneB-Light.otf` (font-weight: 300)
- `StyreneB-Regular.woff2` or `StyreneB-Regular.otf` (font-weight: 400)
- `StyreneB-Medium.woff2` or `StyreneB-Medium.otf` (font-weight: 500)
- `StyreneB-Bold.woff2` or `StyreneB-Bold.otf` (font-weight: 700)
- `StyreneB-Black.woff2` or `StyreneB-Black.otf` (font-weight: 900)

If font files are missing, the app falls back to Helvetica Neue.
